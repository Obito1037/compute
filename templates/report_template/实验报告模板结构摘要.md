# 实验报告模板结构摘要

来源：用户上传的《实验报告封面及正文目录结构.doc》。本摘要用于 Codex 和报告撰写 skill 读取。

## 封面

- 课 程 实 验 报 告
- 课程名称：计算机系统基础
- 专业班级
- 学号
- 姓名
- 指导教师
- 报告日期
- 计算机科学与技术学院

## 正文结构

- 目录
- 实验1
  - 1.1 实验概述
  - 1.2 实验内容
  - 1.3 实验设计
  - 1.4 实验过程
  - 1.5 实验结果
  - 1.6 实验小结
- 实验2/实验3等阶段性实验
  - x.1 实验概述
  - x.2 实验内容
    - x.2.1 阶段1
      1. 任务描述
      2. 实验设计
      3. 实验过程
      4. 实验结果
  - x.3 实验小结
- 实验总结：全面总结实验成果，描述实验收获，400 字以上。

## 本项目扩展

实际实验数量为 5 个，按照相同结构扩展为实验1至实验5。
