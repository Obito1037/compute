# 原始材料说明

本目录保存教师下发的原始实验材料以及已解压文件，供 Codex 和人工阅读使用。

## 目录说明

- `requirements/`：实验任务书、报告任务说明等原始文档。
- `source_archives/`：原始 `.zip` / `.rar` 压缩包，完整保留，不修改。
- `extracted/`：已按实验分类解压的源码、指导文档、项目文件。
- `parsed_text/`：从 docx / pptx 中抽取的纯文本，便于 Codex 快速阅读。

## 使用原则

1. 原始材料只能作为输入资料，不应在其中直接修改。
2. 实验时应把需要操作的源码复制到 `experiments/<lab>/src/` 中。
3. 报告中的结论必须来自实际运行、调试、分析和保存的证据文件。
