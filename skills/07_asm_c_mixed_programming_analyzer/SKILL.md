# C/ASM 混合编程分析器

## 触发场景

当处理 Visual Studio、MASM、C 调用汇编、Debug/Release 对比和性能优化时触发。

## 输入

- 当前实验目录 `experiments/<lab>/`
- 原始材料 `original_materials/`
- 项目规则文件 `01_PROJECT_RULES.md`

## 工作流程

1. 检查 VS/MASM 配置记录。
2. 分析 C 与汇编函数接口。
3. 记录 Debug/Release 性能与反汇编差异。
4. 记录汇编优化和排序优化过程。

## 输出

- C/ASM 接口说明
- 性能表
- Debug/Release 对比
- 优化说明

## 禁止事项

- 不得编造未运行的命令、未保存的输出、未登记的截图。
- 不得直接把推测写成实验事实。
- 不得修改 `original_materials/` 中的原始材料。
