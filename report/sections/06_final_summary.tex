\section*{实验总结}
\addcontentsline{toc}{section}{实验总结}

本节应全面总结计算机系统基础实验的成果，描述通过实验得到的收获，正式提交版本不少于 400 字。建议从以下方面展开：

\begin{enumerate}[label=\arabic*.]
    \item 对数据表示、位级运算、结构体内存布局和浮点表示的理解。
    \item 对 C 程序、汇编程序、函数调用、寄存器和栈帧关系的理解。
    \item 对 GDB 调试、反汇编阅读、控制流分析和二进制程序行为推断的理解。
    \item 对缓冲区溢出、返回地址覆盖、栈帧安全和系统安全风险的认识。
    \item 对 ELF 文件结构、符号表、重定位、静态链接和动态链接机制的认识。
    \item 对实验过程中遇到的问题、解决方式和后续改进方向的反思。
\end{enumerate}
