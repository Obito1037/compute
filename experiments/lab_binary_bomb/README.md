# 二进制炸弹实验：机器级语言理解

本目录用于完成该实验的全过程记录、源码操作、输出保存、图表生成和报告材料整理。

## 工作顺序

1. 阅读 `requirement_summary.md`。
2. 在 `src/` 中进行实验操作，禁止直接修改 `original_materials/`。
3. 所有命令写入 `evidence/commands.md`。
4. 所有输出保存到 `outputs/`。
5. 所有截图登记到 `evidence/screenshots_index.md`。
6. 所有结论登记到 `evidence/report_claims_checklist.md`。
7. 最终总报告统一在根目录 `report/` 中生成。
