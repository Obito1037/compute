# 报告模板对齐说明

本项目已接入用户提供的《实验报告封面及正文目录结构.doc》作为正式格式来源。原始模板位于：

- `original_materials/report_templates/实验报告封面及正文目录结构.doc`
- `original_materials/report_templates/实验报告封面及正文目录结构.docx`

## 模板要求摘要

1. 封面标题为“课程实验报告”。
2. 封面字段包括课程名称、专业班级、学号、姓名、指导教师、报告日期、学院名称。
3. 正文前应有目录。
4. 每个实验应包含：实验概述、实验内容、实验设计、实验过程、实验结果、实验小结。
5. 如果实验按阶段展开，应在实验内容下按阶段描述，每个阶段包含：任务描述、实验设计、实验过程、实验结果。
6. 文末应有“实验总结”，全面总结实验成果和收获，正文要求不少于 400 字。

## 本项目的适配方式

原始模板示例列出了 3 个实验。本项目根据实际材料扩展为 5 个实验章节：

1. 实验1：数据表示和等效运算
2. 实验2：汇编和 C 的混合编程及优化
3. 实验3：二进制炸弹
4. 实验4：缓冲区溢出炸弹
5. 实验5：ELF 文件与程序链接

扩展时保留原模板的核心层级：

- `x.1 实验概述`
- `x.2 实验内容`
- `x.3 实验设计`
- `x.4 实验过程`
- `x.5 实验结果`
- `x.6 实验小结`

对阶段性实验，允许在 `x.2 实验内容` 下继续展开 `x.2.1 阶段1`、`x.2.2 阶段2` 等小节，每个阶段必须说明任务描述、实验设计、实验过程、实验结果。

## 强约束

正式报告正文不得写入 AI 使用说明。AI 使用记录只保存在 `journals/ai_assistance_log.md`。
